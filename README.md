## 🚀 快速开始

### 环境要求

- **R**: ≥ 4.3
- **Quarto**: ≥ 1.4
- **Git**: 用于版本控制（可选）
- **浏览器**: 现代 Web 浏览器（推荐 Chrome/Firefox）

### 本地运行

```bash
# 1. 克隆仓库
git clone https://github.com/KangWang42/R_note_for_Epidemiology.git
cd R_note_for_Epidemiology

# 2. 进入文档目录
cd doc

# 3. 安装依赖（首次运行）
# 确保 R 和 Quarto 已安装
# quarto install tinytex (Windows 用户可选)

# 4. 运行预览
quarto preview

# 5. 构建静态网站
quarto render
```
