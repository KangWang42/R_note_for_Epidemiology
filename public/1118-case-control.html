方法背景与适用场景

病例对照研究(Case-Control Study)是一种回顾性观察性研究设计,从已患病的病例和未患病的对照中,回顾性调查暴露因素,通过比较两组暴露比例来评估暴露与疾病的关联。

适用场景

  • 罕见疾病研究:患病率极低,队列研究样本量需求过大
  • 病因探索:多种暴露因素的初步筛查
  • 疾病暴发调查:快速识别危险因素
  • 长潜伏期疾病:癌症等需长期随访的疾病
  • 资源有限:相比队列研究成本低、周期短

不适用场景

  • 发病率估计:无法计算发病率,只能估计OR
  • 罕见暴露:暴露太少,难以有足够病例
  • 时序不明:回顾性调查难以确定时间先后

与其他研究设计的对比

研究设计 选择对象 比较内容 主要指标 成本 罕见病
病例对照 从结局选择 暴露比例 OR 低 ✓
队列研究 从暴露选择 发病率 RR/HR 高 ✗
横截面 随机抽样 患病率 PR 低 ✗

核心概念与模型入门

通俗理解:用日常例子解释病例对照研究

💡 想象你要调查某种罕见疾病的病因

病例对照研究就像”倒推侦探”: - 找到20个得了这种罕见病的人(病例组) - 再找20个没得病的人(对照组) - 调查两组人过去10年是否接触过某种化学物质 - 发现:病例组80%接触过,对照组只有20%接触过 - 推断:接触该化学物质可能是致病因素 - 这就是”从结局回溯暴露”的逻辑

回到统计学:病例对照研究从”果”到”因”,先确定疾病状态,再回顾暴露历史,适合罕见病和病因探索。

核心术语定义

病例(Cases)

符合特定疾病诊断标准的患者:

  • 新发病例(Incident Cases):近期新诊断,优于现患病例
  • 现患病例(Prevalent Cases):现存病例,可能有生存偏倚

对照(Controls)

未患该病但在其他方面与病例相似的个体:

  • 人群对照:从源人群随机抽取
  • 医院对照:从同一医院其他疾病患者中选择
  • 邻居/朋友对照:与病例生活环境相似

匹配(Matching)

让病例和对照在某些特征上保持一致,以控制混杂:

匹配类型 比例 适用场景
1:1匹配 每个病例匹配1个对照 病例数较多
1:N匹配 每个病例匹配多个对照 病例稀少
频数匹配 组间分布相似,不一一对应 大样本

常见匹配变量:年龄、性别、地区、诊断时间

比值比(Odds Ratio, OR)

病例对照研究的主要效应量:

\[OR = \frac{a/c}{b/d} = \frac{ad}{bc}\]

其中:

           Disease+  Disease-
Exposed+      a         b
Exposed-      c         d

OR的解释: - OR = 1:暴露与疾病无关 - OR > 1:暴露可能是危险因素 - OR < 1:暴露可能是保护因素

罕见病假设:当疾病罕见(<10%)时,OR ≈ RR

病例对照研究的核心特征

1. 回顾性调查

时间轴: 
过去                  现在
────暴露历史────│─────选择病例和对照─────▶
                ↑
            从此时点回溯

2. 选择病例和对照的原则

病例选择: - 明确诊断标准(金标准或临床诊断) - 新发病例优于现患病例 - 来源于明确的源人群

对照选择: - 来自与病例相同的源人群 - 如果对照患病,会被诊断为病例 - 在暴露上具有代表性

3. 回忆偏倚

病例可能比对照更认真回忆暴露历史:

  • 差异性错分:病例过度报告暴露
  • 应对策略:使用客观记录(病历、检测结果)

模型假设与前提条件

假设1:对照代表源人群

含义:对照的暴露分布能代表产生病例的源人群

违背后果:选择偏倚

应对策略: - 人群对照(随机抽样) - 多来源对照(医院+社区)

假设2:暴露测量无偏

含义:病例和对照的暴露测量准确性相同

违背后果:信息偏倚(回忆偏倚)

应对策略: - 盲法调查 - 客观测量 - 验证自我报告

假设3:无严重混杂

含义:已调整主要混杂因素

应对策略: - 匹配 - 分层分析 - 多变量Logistic回归

数据准备

数据结构

非匹配设计:

ID | Case | Exposure | Age | Gender | ...
1  |  1   |    1     | 45  |   0    | ...
2  |  0   |    0     | 48  |   0    | ...

匹配设计:

MatchID | Case | Exposure | Age | Gender | ...
   1    |  1   |    1     | 45  |   0    | ...
   1    |  0   |    0     | 45  |   0    | ...

模拟病例对照数据

set.seed(2026)
library(tidyverse)

n_cases <- 100
n_controls <- 100

# 病例组
cases <- data.frame(
  id = 1:n_cases,
  case = 1,
  age = rnorm(n_cases, 55, 10),
  gender = rbinom(n_cases, 1, 0.6)
)

# 暴露概率(病例组更高)
cases$exposure <- rbinom(n_cases, 1, 0.7)

# 对照组
controls <- data.frame(
  id = (n_cases + 1):(n_cases + n_controls),
  case = 0,
  age = rnorm(n_controls, 52, 10),
  gender = rbinom(n_controls, 1, 0.5)
)

# 暴露概率(对照组更低)
controls$exposure <- rbinom(n_controls, 1, 0.3)

# 合并
case_control_data <- bind_rows(cases, controls) |>
  mutate(
    case_f = factor(case, levels = 0:1, labels = c("对照", "病例")),
    exposure_f = factor(exposure, levels = 0:1, labels = c("未暴露", "暴露")),
    gender_f = factor(gender, levels = 0:1, labels = c("女", "男"))
  )

library(MatchIt)

# 使用年龄和性别匹配
matched <- matchit(
  case ~ age + gender,
  data = case_control_data,
  method = "nearest",
  ratio = 1

# 提取匹配后的数据
matched_data <- match.data(matched)

# 添加匹配ID
matched_data$match_id <- matched_data$subclass

结果解读与报告

OR的正确解释

正确:“暴露者的疾病odds是未暴露者的X倍”

错误:“暴露者的疾病风险是未暴露者的X倍”(这是RR的解释)

罕见病时:OR ≈ RR,可近似解释为风险比

论文报告示例

本病例对照研究纳入100例新诊断的X病患者和100例对照。病例组暴露率为70%,对照组为30%。单因素分析显示,暴露与疾病显著相关(OR=5.44, 95%CI: 3.10-9.55)。

调整年龄和性别后,关联仍显著(调整后OR=4.82, 95%CI: 2.68-8.67),提示暴露者的疾病odds是未暴露者的4.82倍。

常见错误与纠偏

错误1:将OR解释为RR

错误:“暴露使风险增加5倍”

正确:“暴露的OR为5”或”暴露使疾病odds增加5倍”

错误2:匹配变量进入非条件模型

错误做法:

# 匹配了年龄,又在普通Logistic中调整年龄
glm(case ~ exposure + age, data = matched_data, family = binomial)

正确做法:使用条件Logistic

错误3:使用现患病例

问题:生存偏倚,轻症患者已康复

正确:使用新发病例

总结

病例对照研究的优势: 1. 适合罕见病 2. 成本低、周期短 3. 可同时研究多种暴露

局限性: 1. 无法计算发病率 2. 回忆偏倚 3. 选择偏倚

设计要点: 1. 明确病例定义 2. 对照来自源人群 3. 匹配控制混杂

分析要点: 1. 非匹配:普通Logistic回归 2. 匹配:条件Logistic回归 3. OR而非RR
