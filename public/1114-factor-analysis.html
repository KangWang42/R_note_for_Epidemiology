因子分析完全指南

统计分析方法
高级建模
R包
统计模型
因子分析的理论基础与结果解读，包括数学原理、适用性检验、提取方法与旋转策略。
Published

January 25, 2026

简介

因子分析 (Factor Analysis) 是一种多元统计分析方法，用于探索观测变量背后的潜在结构。它假设观测变量受到少数几个不可观测的”因子”（latent factors）影响，通过分析变量间的相关关系来识别这些潜在因子。

核心思想

因子分析的核心思想是：多个观测变量之间的相关性是因为它们共同受少数几个潜在因子的影响。例如，在心理学测试中，多个题目可能都测量同一个潜在特质（如”外向性”）。

EFA vs PCA

特性 因子分析 (EFA) 主成分分析 (PCA)
目标 解释相关性 解释方差
假设 存在潜变量 无潜变量假设
模型 观测 = 载荷 × 因子 + 误差 主成分 = 观测的线性组合
适用场景 量表开发、结构探索 降维、特征提取
因子含义 有理论意义 仅数学组合

数学原理

因子模型

因子分析的基本模型可以表示为：

\[X = \Lambda F + \varepsilon\]

其中： - \(X\)：\(p \times 1\) 观测变量向量 - \(\Lambda\)：\(p \times m\) 载荷矩阵（loadings） - \(F\)：\(m \times 1\) 潜在因子向量 - \(\varepsilon\)：\(p \times 1\) 特异误差（unique error）

直观理解：每个观测变量是因子的线性组合加上特有的误差。

方差分解

在因子分析中，观测变量的总方差可以分解为：

\[\text{总方差} = \text{共同方差} + \text{特异方差}\]

其中： - 共同方差 (Common Variance)：由所有因子共同解释的方差 - 特异方差 (Unique Variance)：因子无法解释的方差，包括测量误差和变量特有变异

相关概念： - 共同度 (Communality, \(h^2\))：某个变量的共同方差占总方差的比例 \[h^2 = \sum_{j=1}^{m} \lambda_{ij}^2\]

其中 \(\lambda_{ij}\) 是变量 \(i\) 在因子 \(j\) 上的载荷

  • 特异度 (Uniqueness, \(u^2\))：某个变量的特异方差占总方差的比例 \[u^2 = 1 - h^2\]

解读： - 共同度高 (>0.6)：变量被因子很好地解释 - 共同度低 (<0.4)：变量可能需要删除或增加更多因子

基本假设

进行因子分析需要满足以下假设：

  1. 样本量充足：每个变量至少 5-10 个观测值
  2. 变量正态性：变量近似服从多元正态分布
  3. 线性关系：变量之间近似线性相关
  4. 变量相关性：变量之间应有足够的相关性（KMO 检验验证）
  5. 无多重共线性：变量间不应过度相关（否则会冗余）

适用性检验

在进行因子分析前，需要检验数据是否适合进行因子分析。

KMO 检验

Kaiser-Meyer-Olkin (KMO) 检验用于衡量变量间偏相关性的程度。

KMO 值的解释：

KMO 值 解释 适用性
>0.9 极好 (Marvelous) 非常适合
0.8-0.9 良好 (Meritorious) 很适合
0.7-0.8 中等 (Middling) 适合
0.6-0.7 一般 (Mediocre) 勉强可以
<0.6 较差 (Unacceptable) 不适合 
# 使用 psych 包进行 KMO 检验
# data("Harman23.cor")  # 加载协方差矩阵
# KMO(Harman23.cor$cov)  # 进行 KMO 检验

Bartlett 球形检验

Bartlett 球形检验用于检验相关矩阵是否为单位矩阵（即变量间是否独立）。

假设： - \(H_0\)：相关矩阵是单位矩阵（变量间无相关） - \(H_1\)：相关矩阵不是单位矩阵（变量间有相关）

解读： - p < 0.05：拒绝 \(H_0\)，变量间存在显著相关，适合进行因子分析 - p ≥ 0.05：不能拒绝 \(H_0\)，变量间相关性不足，不适合进行因子分析

# 使用 psych 包进行 Bartlett 检验
# cortest.bartlett(Harman23.cor$cov, n = Harman23.cor$n.obs)

样本量要求

因子分析需要足够的样本量来获得稳定的结果：

指南 最小样本量
每个变量 5 个观测 \(n \geq 5p\)
每个变量 10 个观测 \(n \geq 10p\)
绝对最小值（不推荐） \(n \geq 100\)

其中 \(n\) 是样本量，\(p\) 是变量数。

注意：样本量不足会导致载荷估计不稳定、共同度偏低等问题。

因子提取方法

主轴因子法

主轴因子法（Principal Axis Factoring, PAF）是最常用的因子提取方法之一。它使用相关矩阵的对角线（共同度）代替原始方差作为初始估计。

特点： - 考虑变量的共同度 - 适合探索性因子分析 - 对小样本较为稳健

# 使用主轴因子法提取因子
fa_pa <- fa(
  x = Harman23.cor$cov,        # 协方差矩阵
  nfactors = 2,                # 提取 2 个因子
  rotate = "varimax",           # 使用 varimax 旋转
  fm = "pa"                    # 主轴因子法
)

最大似然法

最大似然法（Maximum Likelihood, ML）假设数据服从多元正态分布，通过最大化似然函数来估计参数。

特点： - 假设数据服从多元正态分布 - 提供模型拟合统计量 - 适合大样本

# 使用最大似然法提取因子
fa_ml <- fa(
  x = Harman23.cor$cov,
  nfactors = 2,
  rotate = "varimax",
  fm = "ml"                     # 最大似然法
)

提取方法对比

方法 假设 优点 缺点
主轴因子法 无分布假设 对小样本稳健 依赖共同度估计
最大似然法 多元正态分布 有模型拟合检验 对非正态数据敏感

确定因子数量

选择合适的因子数是因子分析的关键步骤。

特征值准则 (Kaiser 准则)

Kaiser 准则建议保留特征值（eigenvalue）大于 1 的因子。

特征值的含义： - 特征值 > 1：因子解释的方差大于一个变量的平均方差 - 特征值 ≤ 1：因子解释的方差小于一个变量的平均方差

# 提取因子时不指定因子数，查看特征值
fa_result <- fa(Harman23.cor$cov, nfactors = 8, rotate = "none")

# 查看特征值
fa_result$values

解读： - 前 n 个因子的特征值 > 1 → 保留 n 个因子 - 特征值下降明显处 → 可能是最佳因子数

碎石图

碎石图（Scree Plot）是特征值按降序排列后的折线图。

使用方法： 1. 查看碎石图的”肘部”（elbow）：曲线变平缓的位置 2. 肘部之前的因子保留，肘部之后的因子舍弃

# 绘制碎石图
library(factoextra)

fviz_screeplot(
  fa_result,
  addlabels = TRUE,                # 添加特征值标签
  main = "碎石图",
  xlab = "因子数",
  ylab = "特征值"
) +
  theme_minimal()

平行分析

平行分析（Parallel Analysis）是最可靠的因子数确定方法之一。

基本思想： 1. 生成与原数据维度相同的随机数据 2. 对随机数据进行因子分析 3. 比较实际数据的特征值与随机数据的特征值 4. 保留实际特征值大于随机数据特征值的因子

# 使用 psych 包进行平行分析
fa.parallel(
  Harman23.cor$cov,             # 协方差矩阵
  fa = "fa",                     # 因子分析
  fm = "pa",                     # 主轴因子法
  n.obs = Harman23.cor$n.obs,    # 样本量
  main = "平行分析"
)

解读： - 实际特征值 > 随机特征值（实线）：保留该因子 - 实际特征值 ≤ 随机特征值：舍弃该因子

因子旋转

因子旋转的目的是提高结果的可解释性，使因子载荷矩阵具有更清晰的结构。

正交旋转 vs 斜交旋转

特性 正交旋转 斜交旋转
因子间相关 因子不相关 (\(\text{cor}(F_i, F_j) = 0\)) 因子可以相关 (\(\text{cor}(F_i, F_j) \neq 0\))
几何解释 因子轴相互垂直 因子轴不一定垂直
适用场景 理论上因子应独立 理论上因子可能相关
方法示例 varimax promax, oblimin

旋转方法选择

Varimax 旋转（正交） - 最常用的旋转方法 - 简化因子载荷矩阵，使每个变量尽可能仅在一个因子上有高载荷 - 适合因子间应该独立的情况

# Varimax 旋转
fa_varimax <- fa(
  Harman23.cor$cov,
  nfactors = 2,
  rotate = "varimax"             # 正交旋转
)

Promax 旋转（斜交） - 快速的斜交旋转 - 允许因子间相关 - 计算效率高

# Promax 旋转
fa_promax <- fa(
  Harman23.cor$cov,
  nfactors = 2,
  rotate = "promax"              # 斜交旋转
)

Oblimin 旋转（斜交） - 更灵活的斜交旋转 - 可以控制因子间相关性的程度 - 适合复杂的数据结构

# Oblimin 旋转
fa_oblimin <- fa(
  Harman23.cor$cov,
  nfactors = 2,
  rotate = "oblimin"            # 斜交旋转
)

旋转选择建议： - 如果理论认为因子应独立 → 正交旋转 (varimax) - 如果理论认为因子可能相关 → 斜交旋转 (promax, oblimin) - 如果不确定 → 可以尝试两种旋转，比较结果的可解释性

结果解读

统计解读

因子载荷

因子载荷（Factor Loadings, \(\lambda_{ij}\)）表示变量 \(i\) 与因子 \(j\) 之间的相关系数。

载荷阈值解读：

载荷绝对值 \(\vert\lambda\vert\) 解释
> 0.7 优秀
0.6-0.7 良好
0.4-0.6 可接受
< 0.4 考虑删除该变量

示例：如果变量 A 在因子 1 上的载荷为 0.8，在因子 2 上的载荷为 0.1，说明变量 A 主要受因子 1 影响。

共同度

共同度（Communality, \(h^2\)）表示某个变量被所有因子共同解释的方差比例。

\[h^2 = 1 - u^2\]

其中 \(u^2\) 是特异度（unique variance）。

共同度解读：

共同度 解释 建议
> 0.6 良好 变量被因子很好地解释
0.4-0.6 可接受 变量被因子适度解释
< 0.2 较差 变量被因子解释不足，考虑删除或增加因子

特异度

特异度（Uniqueness, \(u^2\)）表示某个变量特有的方差（因子无法解释的方差），包括测量误差和变量特有变异。

\[u^2 = 1 - h^2\]

特异度解读： - 特异度高 (>0.8)：变量主要由测量误差或特有变异组成 - 特异度低 (<0.4)：变量主要由因子解释

临床/实际解读

因子命名

根据在因子上载荷高的变量来命名因子。

示例（Harman23.cor 数据集）： - 因子 1：在身高、臂展、前臂、腿长上载荷高 → “体型/身高因子” - 因子 2：在体重、胸围、胸宽上载荷高 → “体型/胖瘦因子”

命名原则： 1. 查看每个因子上载荷最高的几个变量 2. 总结这些变量的共同特征 3. 给出有意义的因子名称 4. 因子名称应与研究领域相关

变量归类

根据载荷矩阵将变量归类到相应的因子。

归类原则： - 如果变量在某个因子上的载荷绝对值 > 0.3，认为该变量属于该因子 - 变量可以在多个因子上有高载荷（特别是在斜交旋转中）

诊断解读

模型拟合

整体拟合： - 所有因子的共同度平均值 - 所有因子解释的总方差比例

# 查看模型拟合指标
fa_result$TL    # 总方差解释比例
fa_result$RMSR  # 均方根残差

诊断： - 总方差解释比例 > 0.6：模型拟合良好 - 均方根残差 < 0.05：模型拟合良好

问题诊断

问题 症状 解决方法
共同度过低 多个变量 \(h^2 < 0.4\) 增加因子数或删除低共同度变量
Heywood 案例 \(h^2 > 1\) 或 \(u^2 < 0\) 检查数据问题、使用不同的提取方法
因子间相关过高 斜交旋转中因子相关 > 0.8 考虑减少因子数
因子难以命名 载荷分布均匀 尝试不同的旋转方法

最佳实践

EFA vs PCA 选择指南

问题 选择 EFA 选择 PCA
是否有理论假设？ 是 → EFA 否 → PCA
是否关注潜在变量？ 是 → EFA 否 → PCA
是否需要可解释的因子？ 是 → EFA 否 → PCA
目标是降维还是探索？ 探索结构 → EFA 降维压缩 → PCA
数据来源是什么？ 问卷、测试 → EFA 任意数据 → PCA

提取方法选择

方法 适用场景 优点 缺点
主轴因子法 (PA) 探索性分析、小样本 稳健、适合共同度低的数据 依赖共同度初始估计
最大似然法 (ML) 数据服从多元正态、大样本 提供模型拟合检验 对非正态数据敏感
最小残差法 (Minres) 通用场景 通常是最佳默认选择 解释不如 PA 和 ML 直观

选择建议： - 首先尝试默认方法（minres） - 如果数据服从正态分布 → 尝试 ML - 如果样本量较小 → 使用 PA

旋转方法选择

情景 推荐旋转
理论认为因子应独立 varimax (正交)
理论认为因子可能相关 promax 或 oblimin (斜交)
不确定 依次尝试 varimax 和 promax，比较结果
需要快速结果 promax
需要灵活控制 oblimin

旋转评估标准： 1. 简洁性：载荷矩阵是否简洁（每个变量主要在一个因子上有高载荷） 2. 可解释性：是否容易给因子命名 3. 理论一致性：旋转结果是否符合理论预期

常见错误与解决

Heywood 案例

问题：某个变量的共同度 > 1 或特异度 < 0（不可能的情况）

原因： - 模型指定过多因子 - 数据存在异常 - 样本量过小

解决方法： 1. 减少因子数量 2. 检查数据质量（异常值、缺失值） 3. 尝试不同的提取方法（ML 可能更适合） 4. 删除异常变量

共同度过低

问题：多个变量的共同度 < 0.4

原因： - 变量与因子结构无关 - 因子数量不足 - 数据噪声过大

解决方法： 1. 增加因子数量 2. 删除低共同度变量（< 0.2） 3. 检查变量是否与因子理论相关 4. 提高数据质量

过度提取

问题：提取的因子数超过实际需要的数量

原因： - 没有使用平行分析 - 仅依赖 Kaiser 准则（特征值 > 1） - 数据噪声被误认为因子

解决方法： 1. 使用平行分析（最可靠的方法） 2. 结合碎石图的肘部判断 3. 理论知识指导因子数选择

因子难以命名

问题：载荷分布均匀，无法找到清晰的因子含义

原因： - 因子数量不合适 - 数据结构复杂 - 旋转方法选择不当

解决方法： 1. 尝试不同的旋转方法（varimax vs promax） 2. 减少或增加因子数 3. 删除不清晰变量 4. 重新考虑变量选择
