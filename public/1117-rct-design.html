方法背景与适用场景

随机对照试验(Randomized Controlled Trial, RCT)是评价干预措施效果的金标准,通过随机分配受试者到干预组或对照组,并在相同条件下进行观察和比较,从而获得因果关系的最强证据。

适用场景

  • 药物疗效评价:新药vs安慰剂或标准治疗
  • 治疗方案比较:手术vs药物治疗
  • 预防措施评估:疫苗、筛查项目
  • 行为干预:生活方式改变、教育项目
  • 诊断方法评价:新检测方法vs金标准

不适用场景

  • 伦理限制:有害暴露(吸烟、辐射)
  • 罕见疾病:难以招募足够样本
  • 成本极高:长期随访、大样本需求
  • 可行性差:干预难以标准化

与其他研究设计的对比

研究设计 随机分配 因果推断 偏倚控制 成本 外推性
RCT 是 最强 最好 很高 有限
队列研究 否 强 好 高 好
病例对照 否 中 中 中 好
横截面 否 弱 差 低 好

核心概念与模型入门

通俗理解:用日常例子解释RCT

💡 想象你要测试一种新的学习方法是否有效

RCT就像”抛硬币分组做实验”: - 招募100个学生,抛硬币决定谁用新方法、谁用传统方法 - 因为是随机分配,两组学生的聪明程度、努力程度都差不多 - 一个学期后考试,新方法组平均85分,传统组平均75分 - 因为唯一的区别就是学习方法,所以可以说:新方法使成绩提高10分 - 这就是”随机化消除混杂”的威力

回到统计学:随机分配使得除了干预措施外,两组在所有已知和未知因素上都均衡,从而能够推断因果关系。

核心术语定义

随机化(Randomization)

通过随机方法(如计算机生成随机数)将受试者分配到不同组别,使每个受试者被分配到任一组的概率相等或预先设定。

随机化的目的: 1. 平衡已知混杂因素 2. 平衡未知混杂因素 3. 消除选择偏倚 4. 为统计推断提供基础

盲法(Blinding/Masking)

隐瞒治疗分配信息,以减少主观偏倚:

类型 不知道分组的人 应用场景
单盲 受试者 安慰剂对照
双盲 受试者+研究者 药物试验(金标准)
三盲 受试者+研究者+数据分析者 高质量试验
开放标签 无人设盲 手术vs药物

分配隐藏(Allocation Concealment)

在随机化后,确保研究者无法预知或影响下一个受试者的分组:

  • 中央随机化:电话/网络系统分配
  • 密封信封:不透光、按序号使用
  • 药物编码:外观相同的药瓶

意向性分析(Intention-to-Treat, ITT)

按照随机分配的组别进行分析,无论受试者是否实际接受了干预:

随机分配100人到干预组
→ 其中10人未接受干预
→ ITT分析仍将这10人算作干预组

符合方案分析(Per-Protocol, PP)

仅分析完全遵循试验方案的受试者:

随机分配100人到干预组
→ 其中10人未接受干预
→ PP分析仅分析实际接受干预的90人

ITT vs PP:

特征 ITT PP
优点 保留随机化优势,避免偏倚 估计真实疗效
缺点 可能低估疗效 可能引入偏倚
适用 主要分析(首选) 敏感性分析

RCT的核心原则

1. 随机化平衡混杂

混杂因素C影响X和Y:
  
观察性研究: C → X → Y (混杂)
            ↓
            Y
            
RCT:  随机化 → X → Y (无混杂)
      (C被平衡)

2. 盲法减少偏倚

  • 表现偏倚:知道分组影响行为
  • 测量偏倚:知道分组影响评估
  • 失访偏倚:知道分组影响退出

3. 对照组的选择

对照类型 适用场景 优缺点
安慰剂 无标准治疗 最强证据,但有伦理争议
标准治疗 已有有效治疗 伦理可接受
剂量对照 最佳剂量不明 可比性好
等待对照 资源有限 伦理可接受

模型假设与前提条件

假设1:随机化成功

含义:随机分配使组间基线特征均衡

检验方法:基线特征表比较(但注意:组间差异p>0.05不等于平衡)

违背后果:混杂偏倚

应对策略: - 使用分层随机化 - 多变量调整 - 敏感性分析

假设2:依从性良好

含义:受试者按方案接受干预

违背后果:稀释效应,低估真实疗效

应对策略: - 提高依从性(教育、激励) - ITT分析保守估计 - PP分析敏感性检验

假设3:失访随机

含义:失访与治疗效果无关

违背后果:选择偏倚

应对策略: - 最小化失访(<10%) - 多重插补 - 敏感性分析(最坏/最好情景)

模拟RCT数据

set.seed(2026)
library(tidyverse)

n <- 200

# 基线特征
rct_data <- data.frame(
  id = 1:n,
  age = rnorm(n, 50, 10),
  gender = rbinom(n, 1, 0.5),
  baseline_sbp = rnorm(n, 140, 15)  # 基线收缩压
)

# 随机分配(1:1比例)
rct_data$group <- sample(0:1, n, replace = TRUE)

# 模拟依从性(干预组85%依从,对照组95%依从)
rct_data$compliance <- ifelse(
  rct_data$group == 1,
  rbinom(n, 1, 0.85),
  rbinom(n, 1, 0.95)
)

# 模拟随访结局(干预组降压效果-10 mmHg)
treatment_effect <- -10
rct_data$followup_sbp <- rct_data$baseline_sbp +
  rnorm(n, mean = 0, sd = 10) +  # 自然波动
  rct_data$group * rct_data$compliance * treatment_effect  # 治疗效应

# 模拟失访(随机5%)
rct_data$dropout <- rbinom(n, 1, 0.05)
rct_data$followup_sbp[rct_data$dropout == 1] <- NA

# 转换为因子
rct_data <- rct_data |>
  mutate(
    group_f = factor(group, levels = 0:1, labels = c("对照组", "干预组")),
    gender_f = factor(gender, levels = 0:1, labels = c("女", "男")),
    compliance_f = factor(compliance, levels = 0:1, labels = c("不依从", "依从"))
  )

# 计算各阶段人数
consort_summary <- data.frame(
  Stage = c("筛选", "随机分配", "干预组分配", "对照组分配",
            "干预组完成", "对照组完成", "干预组分析", "对照组分析"),
  N = c(
    250,  # 筛选
    200,  # 符合条件并随机
    sum(rct_data$group == 1),  # 干预组
    sum(rct_data$group == 0),  # 对照组

论文报告示例

本随机对照试验共纳入200名高血压患者,按1:1比例随机分配到干预组(n=X)或对照组(n=X)。基线特征两组均衡(表1)。

意向性分析显示,干预组随访收缩压较基线下降X.X mmHg (95%CI: X.X-X.X),对照组下降X.X mmHg (95%CI: X.X-X.X)。组间差异为X.X mmHg (95%CI: X.X-X.X, P<0.001),提示干预显著降低收缩压。

符合方案分析显示更大的治疗效应(X.X mmHg, 95%CI: X.X-X.X),提示依从性影响疗效。

常见错误与纠偏

错误1:基线不平衡就调整

错误做法:发现基线p<0.05,就调整该变量

正确做法: - RCT不需要检验基线差异 - 预先指定调整变量(无论p值) - 基于临床考虑,非统计显著性

错误2:仅报告PP分析

错误做法:因为ITT稀释效应,只报PP

正确做法: - ITT为主要分析 - PP作为敏感性分析

错误3:排除失访者

错误做法:完全病例分析

正确做法: - 报告失访率和原因 - 多重插补 - 敏感性分析(最坏/最好情景)

总结

RCT的优势: 1. 随机化消除混杂,因果推断最强 2. 盲法减少偏倚 3. 前瞻性设计,数据质量高

局限性: 1. 成本高、周期长 2. 外推性有限(严格纳入标准) 3. 伦理限制

设计要点: 1. 随机化方法(序列生成、分配隐藏) 2. 盲法实施(单盲/双盲/三盲) 3. 样本量充足(power≥80%)

分析要点: 1. ITT为主要分析 2. 调整基线协变量 3. PP作为敏感性分析 4. 缺失数据处理
