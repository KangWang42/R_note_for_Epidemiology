# R 语言学习笔记

一个全面、系统的 R 语言数据科学学习资源库，覆盖数据分析方法、实用 R 包教程与可视化复现方案，面向从入门到进阶的数据科学从业者。

## 项目概览

**网站地址**: [https://r.wk8686.top](https://r.wk8686.top)

### 项目特色

- **系统化教程**: 从基础语法到高级统计建模，循序渐进。
- **代码可复现**: 每篇教程都配有完整可运行的代码示例。
- **实战导向**: 以真实数据分析场景为背景编写。
- **持续更新**: 紧跟 R 生态系统的最新发展。

## 内容导航

### 实用 R 包

| 包名 | 用途 | 教程内容 |
| :--- | :--- | :--- |
| **bruceR** | 一站式统计分析 | 效应量、信效度、方差分析 |
| **compareGroups** | 描述性统计表 | 自动生成基线表 |
| **scitb** | 快速基线表 | 期刊级别表格输出 |
| **purrr** | 函数式编程 | 批量数据处理 |
| **future** | 并行计算 | 多核加速任务执行 |
| **data.table** | 高效数据处理 | 大数据集操作 |

### 数据可视化

- **基础图表**: 折线图与时间序列、面积图与堆叠图、柱状图与饼图、箱线图与小提琴图、散点图与趋势线、直方图与密度图。
- **专业图表**: 热图 (Heatmap)、森林图 (Forest Plot)、桑基图 (Sankey)、雷达图、韦恩图、地图可视化。
- **高级技巧**: ggplot2 配色方案、patchwork 多图拼接、cowplot 图形组合、双坐标轴、图例自定义、复杂组合图与发表级图表。

### 统计分析方法

- **回归分析**: Logistic 回归（二分类结局）、多元线性回归（连续结局）、Poisson/负二项回归（计数数据）、广义加性模型 (GAM)。
- **因果推断**: 倾向性得分匹配 (PSM)、双重差分法 (DiD)、中介效应分析。
- **纵向数据**: 生存分析 (KM/Cox)、多水平/混合效应模型、时间序列分析。
- **循证医学**: Meta 分析（效应量合并/网状 Meta）、伞状综述、范围综述。
- **高级方法**: 结构方程模型 (SEM)、因子分析 (EFA/CFA)、主成分分析 (PCA)、潜类别分析 (LCA)、限制性立方样条 (RCS)、变分模态分解 (VMD)。

### 开发环境

- **IDE 指南**: Positron IDE 完全指南（下一代数据科学 IDE）。
- **工具教程**: RMarkdown 入门、Quarto vs RMarkdown 选型对比。

## 项目结构

```text

├── doc/                  # Quarto 源文件目录
│   ├── index.qmd        # 网站首页

│   └── _quarto.yml      # Quarto 配置

└── README.md            # 项目说明
```

## 本地运行

```bash
# 1. 克隆仓库
git clone https://github.com/KangWang42/R_note_for_Epidemiology.git
cd R_note_for_Epidemiology

# 2. 进入文档目录
cd doc

# 3. 运行预览 (需安装 R 和 Quarto)
quarto preview

# 4. 构建静态网站
quarto render
```
