# R 语言学习笔记

一个全面、系统的 R 语言数据科学学习资源库。这里汇集了数据分析方法、实用 R 包教程以及精美可视化图表的复现方案，旨在帮助从入门到进阶的数据科学从业者掌握 R 语言的核心技能。

## 在线访问

**网站地址**: [https://r.wk8686.top](https://r.wk8686.top)

## 项目特色

- **系统化教程**: 从基础语法到高级统计建模，循序渐进
- **代码可复现**: 每篇教程都配有完整可运行的代码示例
- **实战导向**: 以真实数据分析场景为背景编写
- **持续更新**: 紧跟 R 生态系统的最新发展

## 内容导航

### 实用 R 包

| 包名 | 用途 | 教程 |
|------|------|------|
| bruceR | 一站式统计分析 | 效应量、信效度、方差分析 |
| compareGroups | 描述性统计表 | 自动生成基线表 |
| scitb | 快速基线表 | 期刊级别表格输出 |
| purrr | 函数式编程 | 批量数据处理 |
| future | 并行计算 | 多核加速任务执行 |
| data.table | 高效数据处理 | 大数据集操作 |

### 数据可视化

**基础图表**
- 折线图与时间序列、面积图与堆叠图、柱状图与饼图
- 箱线图与小提琴图、散点图与趋势线、直方图与密度图

**专业图表**
- 热图 (Heatmap)、森林图 (Forest Plot)、桑基图 (Sankey)
- 雷达图、韦恩图、地图可视化

**高级技巧**
- ggplot2 配色方案、patchwork 多图拼接、cowplot 图形组合
- 双坐标轴、图例自定义、复杂组合图与发表级图表

### 统计分析方法

**回归分析**
- Logistic 回归 - 二分类结局分析
- 多元线性回归 - 连续结局建模
- Poisson/负二项回归 - 计数数据分析
- 广义加性模型 (GAM) - 非线性关系

**因果推断**
- 倾向性得分匹配 (PSM) - 观察性研究偏倚控制
- 双重差分法 (DiD) - 政策效果评估
- 中介效应分析 - 作用机制探索

**纵向数据**
- 生存分析 - Kaplan-Meier 与 Cox 回归
- 多水平/混合效应模型 - 嵌套数据结构
- 时间序列分析 - 趋势预测

**循证医学**
- Meta 分析 - 效应量合并
- 网状 Meta 分析 - 多干预比较
- 伞状综述 - 证据分级
- 范围综述 - 文献地图

**高级方法**
- 结构方程模型 (SEM) - 潜变量路径分析
- 因子分析 (EFA/CFA) - 量表开发验证
- 主成分分析 (PCA) - 降维与特征提取
- 潜类别分析 (LCA) - 亚组识别
- 限制性立方样条 (RCS) - 剂量反应关系
- 变分模态分解 (VMD) - 信号处理

## 本地运行

```bash
# 克隆仓库
git clone https://github.com/KangWang42/R_note_for_Epidemiology.git
cd R_note_for_Epidemiology

# 进入文档目录
cd doc

# 安装依赖（首次运行）
# 确保已安装 R 和 Quarto

# 预览网站
quarto preview

# 构建静态网站
quarto render
```
